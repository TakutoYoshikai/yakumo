# yakumo
yakumo is a steganography tool.

### Requirements
* macOS
* Python3

### Usage
**install**
```bash
pip3 install git+https://github.com/TakutoYoshikai/yakumo.git
```

**hide data**
```bash
yakumo hide <IMAGE DIR> <SECRET DATA DIR>
```

**reveal**
```bash
yakumo reveal <IMAGE DIR>
```
